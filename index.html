Integritetspolicy

Ullaredaren Shopping App
Senast uppdaterad: September 2025
1. Inledning

Välkommen till Ullaredaren! Jag värnar om din integritet och vill tydligt klargöra att jag inte delar någon personlig eller icke-personlig information när du använder mina tjänster.

🔒 Din integritet är viktig för mig. Jag delar inte din information med tredje part.

2. Datainsamling

När du använder min Ullaredaren-app samlar jag in vissa typer av information. Denna data hjälper mig att förbättra tjänsterna och ge en bättre användarupplevelse.

  • Kontaktinformation: Detta inkluderar detaljer som din e-postadress, som kan användas för kontokonfiguration eller kommunikation.
  • Användarinnehåll: Allt innehåll som du skapar, laddar upp eller delar i appen, som foton, text eller annat material.
  • Identifierare: Unika identifierare, enhets-ID eller andra identifierare som hjälper mig att personalisera din appupplevelse eller spåra appanvändning för analysändamål.
  • Diagnostik: Information relaterad till appprestanda, kraschrapporter och annan diagnostisk data som hjälper mig identifiera och åtgärda tekniska problem.
3. Användning av data

Jag använder insamlad data för att förbättra appens funktionalitet och användarupplevelse. Data används aldrig för reklam eller försäljning till tredje part.

4. Datasäkerhet

Jag tar säkerheten för dina data på allvar och implementerar lämpliga åtgärder för att skydda dem mot obehörig åtkomst eller avslöjande.

5. Ändringar av denna integritetspolicy

Jag förbehåller mig rätten att uppdatera denna integritetspolicy när som helst. Jag kommer att meddela dig om eventuella ändringar genom att publicera den nya integritetspolicyn på denna sida och uppdatera datumet 'Senast uppdaterad'.
